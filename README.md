# Java Developer Roadmap Tracker 🚀

A modern **Java learning roadmap tracker** built with **HTML, CSS, and JavaScript**.
This project helps developers track their progress while learning Java using an interactive roadmap.

It includes features like **checkbox progress tracking, progress bar, search, dark mode, and automatic saving using LocalStorage**.

---

## ✨ Features

* ✅ Interactive Java learning roadmap
* 📊 Progress bar showing completion percentage
* ☑️ Checkbox-based topic tracking
* 💾 Auto-save progress using **LocalStorage**
* 🌙 Dark / Light mode toggle
* 🔍 Search topics instantly
* 📱 Fully responsive layout
* 🎯 Completed topics highlighted automatically

## 🎯 How It Works

* Each topic has a **checkbox**.
* When a checkbox is selected:

  * It gets **marked as completed**
  * The **progress bar updates**
  * The status is saved in **LocalStorage**
* Even after refreshing the page, your progress remains saved.
